// Early Game: Turn 1: Build influence; sponsor TCiM, Sponsor TCiF rotating TCiM to gain another influence. 
// Turn 2: Build (unless you get a Psi Corps card you can sponsor.) Turn 3: Sponsor a character if you have 
// one, otherwise, build. Turn 4: Promote, sponsor or build, depending on what's in your hand, and what you
// did last turn. Turn 5: You should either be at 8 and have another inner circle character, or 9 without 
// anything. If you're at 8, build once, then sponsor Nightwatch Collaborators. Otherwise, build to 9. 
// Turn 6: Hopefully, you're ready to build to 10, and drop TCiM/TCiF, otherwise, build to 10.
// Turn 7: (if you need it.) Sponsor Collaborators, drop TCiM/TCiF and get Kelsey (unless you have her, 
// otherwise, Reprogrammer), Teep Recruiters (otherwise, Psi Corps Intel) and OOPF. Use Collaborators to 
// get the conspiracy marks, they are very necessary.

// Mid Game: Bring out the Characters, making them Nightwatch as fast as is reasonable. Most people won't be 
// interfering for the first 3 or 4, until you start getting them to the inner circle. Win a few conflicts to
// get you influence, so that you're around 14 or 15.

// End Game: Either push through a few more conflicts with your massive intrigue and Earth to gain more influence,
// or Pull out SotM. You should be pulling between 80 and 100 intrigue by this point, so your conflicts will go 
// through easily. Beyond that, virtually every character in the deck can count as a mighty character, so 
// that's potentially 18 power!
