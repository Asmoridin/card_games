// This deck was originally the old Hyach Unification Deck, but after Crusade, it polymorphed well
// into this deck. I was truly dumbfounded by it's ability to generate ghost influence, and soon geared
// it more towards that end for the power that goes with Elizabeth Trent.

// Early Game: - Turn 2 should have you with Kullenbrak (your main ambassador) and Fashar in the nner Circle,
// FoP out, 4 influence, and 2 ghost influence. You at ten real influence at the end of turn 5, after which
// you should sponsor Melat and start pumping out characters.

// Middle Game: Swarm out characters (all at -2 cost), promote ambassadors for the extra ghost, and get Trent
// to the inner circle. When you are ready, win Record Returns (using Aides and Trent's power to push you over)
// and reap the ghost influence. Keep getting out more power. Win a vote or two by tiebreaker (using Backroom Dealing
// and Call in a Marker) to get permanent power from Fulcrum. Save Rule by the Masses when you find it for when you
// can immediately replace it again with Fulcrum after using the rule to draw tons of cards.

// End Game: Sheer Power, with safety in numbers to solidify the sure victory. Steam roll with extra influence you
// really can't use except for Trent's power.

// Other Comments: This is a truly scary deck. Non-Aligned blends perfectly with Corporate themes. I'm still
// not sure this theme works well with other races yet though...
