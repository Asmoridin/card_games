// This was started as a G'Kar deck, but the new version of Na'Far allows me to do evil things with
// the characters of other races that work well for attacks (like Draal). Besides, G'Kar Enlightned
// wouldn't let me use my Strife, so this change worked out well.

// Early Game: Play it all first turn (UTF, get influence, replace, then support with the now ready CN).
// Then try to get up a character ASAP (I usually go for something cheap, like an Aide or Agitator.
// You're strong first off, so no one will mess with you ASAP, but having others helps.

// Middle Game: Try to find your key cards (attackers, Thenta Makur, etc) and promote your attackers.
// Head hunt as soon as you find one of the Seizing Advantages. Get that power, and try to win a conflict
// or two. Make people fear you. I've won an Affirmation of Power here with 10 support to no opposing,
// with Tu'Pari and Draal each with two strife marks as enforcers. This is what you want.

// End Game: Win conflicts while you must, get some head hunting power, and keep pushing through
// assassins. You can't use race specific agenda with CN, so the only thing I could add would be a few Support of the Mightys.
