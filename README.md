# Card Games Index

A comprehensive index of collectible card games (CCGs) that require pre-game deck construction, similar to Magic the Gathering.

## Structure

Games are organized alphabetically by first letter:
- **Folders 0-9**: Games starting with numbers
- **Folders A-Z**: Games organized by first letter (e.g., Magic: The Gathering is in `M/Magic_The_Gathering`)
- **General**: Games that don't fit the alphabetical scheme

## Contents per Game

Each game folder contains:
- `data/` — Game data, card lists, balance information
- `scripts/` — Utilities and tools for the game
- `rules/` — Rulebook and official documentation
- `examples/` — Sample decks or play examples
